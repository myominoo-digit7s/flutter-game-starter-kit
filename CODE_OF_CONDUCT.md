# Code of Conduct

## Our Pledge

We as contributors and maintainers of **Flutter + Flame Game Starter Kit** pledge to make participation in this project a welcoming, respectful, and harassment-free experience for everyone.

This project is intended to help Flutter developers, indie game developers, students, and anyone interested in building 2D games with Flutter and Flame.

We welcome people of all backgrounds, skill levels, and experiences.

## Our Standards

Examples of behavior that contributes to a positive community include:

* Being respectful and considerate of others
* Giving constructive feedback
* Accepting feedback gracefully
* Helping others learn and improve
* Using welcoming and inclusive language
* Focusing on what is best for the project and the community
* Being patient with beginners and new contributors

Examples of unacceptable behavior include:

* Harassment, insults, or personal attacks
* Discriminatory language or behavior
* Trolling, bullying, or intentionally disruptive comments
* Public or private harassment
* Sharing private information without permission
* Spam or unrelated self-promotion
* Any behavior that makes the community unsafe or unwelcoming

## Responsibilities of Maintainers

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior.

Maintainers may take appropriate action in response to unacceptable behavior, including:

* Asking the person to stop the behavior
* Removing comments, issues, pull requests, or discussions
* Temporarily or permanently banning contributors from the project
* Taking other actions necessary to protect the community

Maintainers should apply these standards fairly and consistently.

## Reporting Issues

If you experience or witness unacceptable behavior, please report it to the project maintainer.

You can contact the maintainer privately at:

```text
myomin.oo@digit7s.com
```

Please include as much detail as possible so the issue can be reviewed properly.

All reports will be handled with care and respect. The maintainer will do their best to protect the privacy and safety of the people involved.

## Scope

This Code of Conduct applies to all project spaces, including:

* GitHub issues
* Pull requests
* Discussions
* Code reviews
* Documentation contributions
* Any public or private communication related to this project

It also applies when someone is representing the project in public spaces.

## Enforcement

If someone violates this Code of Conduct, the maintainer may take action depending on the situation.

Possible actions include:

1. A private or public warning
2. Removal of inappropriate content
3. Temporary restriction from participating
4. Permanent ban from the project community

The goal is to keep the project safe, respectful, and helpful for everyone.
